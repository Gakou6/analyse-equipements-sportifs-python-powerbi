# Analyse territoriale des équipements sportifs en France

## Objectif du projet

Ce projet a pour objectif d’analyser la répartition des équipements sportifs en France à partir de données nationales, afin d’identifier des disparités d’accès selon les territoires.

## Contexte

Projet réalisé dans le cadre de ma formation en BUT Science des Données à l’IUT de Lille, site de Roubaix.

L’analyse repose sur plusieurs fichiers de données liés aux installations sportives, aux équipements et aux activités pratiquées. Le projet permet de travailler sur un volume important de données, avec plus de 150 000 lignes traitées.

## Outils utilisés

- Python
- R
- Power BI
- Excel
- Pandas
- Visualisation de données

## Données utilisées

Les données utilisées concernent le recensement des équipements sportifs en France.

Le travail porte notamment sur :

- les installations sportives ;
- les équipements sportifs ;
- les activités pratiquées ;
- les territoires ;
- les indicateurs statistiques par zone géographique.

## Étapes réalisées

1. Importation des fichiers de données
2. Nettoyage et structuration des données
3. Sélection des variables utiles
4. Analyse des équipements sportifs par territoire
5. Création d’indicateurs statistiques
6. Analyse des disparités d’accès aux équipements sportifs
7. Création d’un rapport interactif Power BI

## Aperçu du dashboard

Une capture du rapport Power BI sera ajoutée dans le dossier `visuals`.

## Compétences mises en avant

- Nettoyage de données volumineuses
- Analyse statistique descriptive
- Traitement de fichiers avec Python
- Exploitation de données territoriales
- Création d’indicateurs
- Datavisualisation avec Power BI
- Analyse et interprétation de résultats

## Résultat

Ce projet permet de mieux comprendre la répartition des équipements sportifs sur le territoire français et de mettre en évidence certaines différences d’accès selon les zones géographiques.
