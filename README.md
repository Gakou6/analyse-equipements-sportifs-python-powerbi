# Analyse territoriale des équipements sportifs en France

## Objectif du projet

Ce projet a pour objectif d’analyser la répartition des équipements sportifs en France à partir de données nationales, afin d’identifier des disparités d’accès selon les territoires.

## Contexte

Projet réalisé dans le cadre de ma formation en BUT Science des Données.

L’analyse repose sur des données liées aux installations sportives, aux équipements et aux activités pratiquées. Le projet permet de travailler sur un volume important de données et de produire des indicateurs statistiques par territoire.

## Outils utilisés

- Python
- R
- Power BI
- Excel

## Structure du dépôt

- `data/` : description des données utilisées dans le projet.
- `src/` : scripts Python et R utilisés pour le traitement et l’analyse.
- `visuals/` : captures du rapport Power BI.
- `README.md` : présentation du projet.

## Étapes réalisées

1. Importation des données
2. Nettoyage et structuration des fichiers
3. Analyse des installations, équipements et activités sportives
4. Création d’indicateurs statistiques par territoire
5. Analyse des disparités d’accès aux équipements sportifs
6. Réalisation d’un rapport interactif Power BI

## Aperçu du rapport Power BI

Les captures du rapport sont disponibles dans le dossier `visuals/`.

## Remarque sur le fichier Power BI

Le fichier Power BI complet n’est pas inclus dans ce dépôt en raison de sa taille importante.  
Des captures sont fournies afin de présenter les principaux indicateurs et visualisations réalisés.
